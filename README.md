  **# 3.内部排序**  
  序号12为内部排序算法总结：包含13个子文件。  
所列排序算法有：  
    1) 插入排序：(1)直接插入排序，(2)折半插入排序，(3)2路插入排序，(4)希尔排序。  
    2)交换排序：(1)冒泡排序，(2)快速排序。  
    3)选择排序：(1)直接选择排序，(2)树形选择排序，(3)堆排序。  
    4)归并排序。  
    5)基数排序。  
    6)计数排序；桶排序。
